# fzlzjerry.github.io

This repository hosts the code for my personal GitHub Pages site.

## Features

- **HTML**: The primary language used for structuring the webpage.
- **GitHub Pages**: The site is hosted using GitHub Pages.

## Installation

To run this project locally, clone the repository and open the `index.html` file in your browser:

```sh
git clone https://github.com/fzlzjerry/fzlzjerry.github.io.git
cd fzlzjerry.github.io
open index.html
```
